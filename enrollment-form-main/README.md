# TESDA Application Form

A responsive, professional TESDA (Technical Education and Skills Development Authority) application form built with HTML and CSS only. This form is designed to collect applicant information and can be printed as an official document on A4 paper.

## Features

- **Fully Responsive**: Works seamlessly on desktop, tablet, and mobile devices
- **Professional Layout**: Matches the official TESDA form design with proper formatting
- **Print-Ready**: Optimized for A4 paper printing with clean, document-style output
- **Two-Page Form**: Comprehensive application form split into two pages
- **Auto-Navigation**: Letter boxes automatically advance to the next field
- **Age Calculator**: Automatically calculates age from birthdate
- **Clean Print Output**: Buttons and unnecessary elements are hidden when printing

## Form Sections

### Page 1
1. **Header**: TESDA logo and official title
2. **Reference Number**: Color-coded boxes for tracking
3. **ULI (Unique Learners Identifier)**: Formatted input boxes
4. **School/Training Center Information**
5. **Assessment Type Selection**
6. **Client Type**
7. **Profile Section**:
   - Name (with letter boxes)
   - Mailing Address
   - Parents' Names
   - Personal Details (Sex, Civil Status, Contact Numbers)
   - Educational Attainment
   - Employment Status
   - Birth Date and Place

### Page 2
3. **Work Experience**: Table for employment history
4. **Training/Seminars Attended**: Table for professional development
5. **Competency/Qualification**: Industry sector and NC level
6. **Licensure Examination**: Professional license details
7. **Emergency Contact Information**
8. **Declaration**: Applicant certification
9. **For Office Use Only**: Administrative section

## How to Use

1. **Open the Form**: Open `index.html` in any modern web browser
2. **Fill Out Information**: 
   - Type in the input fields
   - Letter boxes will auto-advance to the next box
   - Check appropriate checkboxes
   - Enter dates in the date fields
3. **Navigate**: 
   - Click "Continue to Page 2" to move to the second page
   - Click "Back to Page 1" to return to the first page
4. **Print**: 
   - Click the "Print Form" button
   - Or use Ctrl+P (Windows) / Cmd+P (Mac)
   - Select A4 paper size in print settings
   - The form will automatically format for professional printing

## Print Features

When printing, the form automatically:
- Hides all buttons and navigation elements
- Removes input field borders for a clean look
- Preserves color-coded boxes with exact colors
- Formats content to fit A4 paper (210mm Ã— 297mm)
- Maintains proper page breaks between sections

1. **Printing**: Use "Print to PDF" for digital submission
2. **Mobile**: Use landscape mode for better visibility of tables
3. **Data Entry**: Fill out completely before printing
4. **Checkboxes**: Only select applicable options
5. **Letter Boxes**: Type one character per box for proper formatting
