# Admin Assets

All assets (CSS, JavaScript, images, fonts) for the admin dashboard application.

## Structure

- `css/` - 16 stylesheet files for different pages and components
- `js/` - 37 JavaScript files for application functionality
- `images/` - Local images and branding assets
- `img/` - Additional UI images and illustrations
- `fonts/` - Font files
- `vendor/` - Third-party libraries

## Asset Organization

Each page typically has its own CSS and JS files for modularity and maintainability.

## Shared Assets

For shared images and branding, reference the `/img/` folder at project root using relative paths: `../../img/[filename]`
