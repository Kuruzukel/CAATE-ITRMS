# Auth Assets

All assets (CSS, JavaScript, images, fonts) for the authentication system.

## Structure

- `css/` - Stylesheets for authentication pages
- `js/` - JavaScript files for authentication functionality
- `images/` - Local images and branding assets
- `fonts/` - Font files (Boxicons)

## Asset Organization

Each authentication page has its own CSS and JS files for modularity.

## Shared Assets

For shared images and branding, reference the `/img/` folder at project root using relative paths: `../../img/[filename]`
